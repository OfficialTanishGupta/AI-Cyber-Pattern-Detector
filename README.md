# 🛡️ AI Cyber Threat Pattern Detector

An advanced cybersecurity AI project built using PyTorch that learns hidden patterns in network traffic and detects anomalous behavior automatically using deep learning.

---

# 🚀 Project Overview

Traditional intrusion detection systems rely on predefined rules and signatures.

This project uses:

- Deep Learning
- Autoencoders
- Pattern Recognition
- Anomaly Detection
- Representation Learning

to identify suspicious network behavior by learning what "normal" traffic looks like.

---

# 🎯 Project Goals

The AI system will:

✅ Learn normal network traffic behavior  
✅ Detect abnormal/suspicious traffic  
✅ Discover hidden traffic patterns  
✅ Visualize network embeddings  
✅ Perform anomaly detection in real-time  
✅ Simulate intrusion detection systems (IDS)

# 🛠️ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Cyber-Pattern-Detector.git
```

---

## 2️⃣ Open Project Folder

```bash
cd AI-Cyber-Pattern-Detector
```

---

## 3️⃣ Create Virtual Environment

```bash
python -m venv venv
```

---

## 4️⃣ Activate Environment

### Windows PowerShell

```powershell
venv\Scripts\activate
```

### Git Bash

```bash
source venv/Scripts/activate
```

---

## 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Dataset Loader

Go to source folder:

```bash
cd src
```

Run:

```bash
python dataset.py
```
